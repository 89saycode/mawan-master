# 分支语句 switch
### switch语句中的关键字
- 表达式
- case n: n代表与表达式进行比较的值。
- break: 表示终止。
- default: 表示所有条件都不符合则执行default中的语句体n+1  


```java
// 格式
switch(关系表达式) {
    case 1:
        语句体1;
        break;
    case 2:
        语句体2;
        break;
    ...
    default:
        语句体n+1;
        break;
}
```
