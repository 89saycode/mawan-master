# 条件语句
## if语句
如果关系表达式为真(true)，则执行语句体。

```java
// if语句格式
if(关系表达式) { 
    语句体;
}
```

## if...else语句
如果关系表达式为真(true),则执行语句体1，否则执行语句体2。
```java
// if...else格式
if(关系表达式) {
    语句体1;
}else {
    语句体2;
}
```

## if...else if...else语句
如果关系表达式1为真(true),则执行语句体1，否则判断关系表达式2为真(true),则执行语句体2，否则执行语句体3。

```java
if (关系表达式1) {
    语句体1;
} else if (关系表达式2) {
    语句体2;
} else {
    语句体3;
}
```
